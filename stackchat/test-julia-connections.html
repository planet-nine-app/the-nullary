Understanding the julia-based connection URL format:

Format: stackchat://connect?message=TIMESTAMP:PUBLICKEY:NAME:JULIAURL&signature=SIG&publicKey=PUB&name=NAME&juliaUrl=URL Components: - message: Contains timestamp, public key, name, and julia server URL - signature: Cryptographic signature of the message - publicKey: User's public key for verification - name: Display name for the connection - juliaUrl: Julia server URL for messaging
